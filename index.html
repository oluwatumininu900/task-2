Used to start a brand-new local Git repository inside a project directory.

Prepares altered files and modified scripts for local saving by copying them to the staging area.

git add .

Recording Project Milestones

Creates isolated development environments to manage code changes cleanly without touching production branches.

git branch feature-documentation
git checkout feature-documentation

Connects local file architectures to remote hosting architectures and securely transfers state tracking timelines online.

git push -u origin main
